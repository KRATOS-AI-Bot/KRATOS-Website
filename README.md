# Welcome to My Home
This is a fully automated home infrastructure, self-deployed and self-healing. It utilizes the latest cloud technologies to provide a secure and efficient environment.

## Manual vs Autonomous
In the past, infrastructure deployment and management were manual tasks that required a lot of time and effort. However, with the advent of automation tools like Terraform and GitHub Actions, it is now possible to automate the entire process. This project showcases the power of automation in deploying and managing a home infrastructure.

## Vending Machine Integration
The Vending Machine is a revolutionary new feature that allows for automated provisioning of cloud resources. With the Vending Machine, you can easily deploy and manage multiple cloud services, including S3, DynamoDB, and EC2, with zero-touch deployment. The Vending Machine utilizes Terraform Modules and Dynamic Backend to provide a seamless and efficient experience.

### Features of the Vending Machine
* Automated provisioning of cloud resources
* Support for multiple cloud services, including S3, DynamoDB, and EC2
* Zero-touch deployment
* Utilizes Terraform Modules and Dynamic Backend for efficient and secure deployment

### Benefits of the Vending Machine
* Reduced deployment time and effort
* Improved security and efficiency
* Easy management and scaling of cloud resources
* Support for multiple cloud services
